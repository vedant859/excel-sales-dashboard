# Excel Sales Dashboard

## Project Overview

Created a sales dashboard in Microsoft Excel to analyze sales data and present key trends through visual reports and charts.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts

## Dashboard Features

* Sales data analysis
* Interactive charts and visualizations
* Trend identification using sales data
* Dashboard-based reporting

## Dataset

* Sales dataset containing approximately 3,000 records

## Skills Demonstrated

* Data Analysis
* Data Visualization
* Dashboard Development
* Excel Reporting

## File Information

Dashboard.xlsx contains the sales dataset and the final dashboard. Supporting sheets used for chart creation and calculations have been hidden to maintain dashboard functionality and improve usability.
